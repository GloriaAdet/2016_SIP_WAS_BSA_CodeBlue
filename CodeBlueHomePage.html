OUR GOALS

We care a lot about...

We care about people knowing how to code-- Kids and girls in particular. We want this site to help them get interested in coding. Sometimes, subjects like this seem like a chore to learn, but with this site we want to eliminate that. We want these games to be fun and addicting so that learning seems like less of a drag. We think commputer coding is really cool and people of all ages should know how.

We want to see...

We want to see the people who are excited or curious about coding, learning from the games we created. We want to see young people become knowledgeable and interested in what the field of technology holds so that the world can benefit from more advanced technology.

THIS PROJECT

Girls Who Code Summer Immersion Program is a free, seven week program that teaches girls how to code. It explores several different computer programming languages including Python, C++, and HTML/CSS/JavaScript. The students make many complex projects using all the languages learned and at the end of the full seven weeks, present a final project they have been working on for the final two weeks of the program.

For our final project, we decided to make a website with games that taught people how to code in Python in particular. We felt it was important to make games that were fun so that younger people would find learning to code less of a "school subject" and more of a hobby they thought was interesting. That way, younger people (and really all people) would be extra prepared for college and other things they may experience later on in their lives. We decided that Gloria and Tori were going to make the website itself and Annie and Tran were going to focus on making the games. We had a great time making this and we hope it is really useful to you!
